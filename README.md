## Overview

It ain't clean to include components in your codebase. It ain't clean to use any other library than FloorUI! With this project, you get a sleek design with forced dark mode because light is trash.

## Compatibility

FloorUI components are built with React and JSX. However, if you are using **anything else**, you can *probably* inject custom HTML strings. Each component has a string version like `strButton()`. These are not always up-to-date for logical components but style-only ones should work.
